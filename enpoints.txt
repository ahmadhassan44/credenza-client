Authentication Endpoints
POST /api/v1/auth/register - Register a new user
POST /api/v1/auth/login - User login
POST /api/v1/auth/logout - User logout (requires JWT)
POST /api/v1/auth/refresh - Refresh access token
GET /api/v1/auth/profile - Get current user profile (requires JWT)
User Endpoints
GET /api/v1/users/profile - Get current user profile (requires JWT)
PATCH /api/v1/users/profile - Update current user profile (requires JWT)
DELETE /api/v1/users/profile - Delete current user account (requires JWT)
GET /api/v1/users/:id - Get user by ID (Admin only)
PATCH /api/v1/users/:id - Update user by ID (Admin only)
DELETE /api/v1/users/:id - Delete user by ID (Admin only)
Platform Endpoints
POST /api/v1/platforms/connect - Connect a platform for a creator (Creator or Admin)
GET /api/v1/platforms - Get all platforms (Admin only)
POST /api/v1/platforms/:id/refresh - Manually refresh metrics for a platform (Creator or Admin)
GET /api/v1/platforms/:id - Get platform by ID (implied from service)
Credit Scoring Endpoints
POST /api/v1/credit-scoring/generate/:creatorId - Generate credit score for a creator (Creator or Admin)
GET /api/v1/credit-scoring/latest/:creatorId - Get latest credit score for a creator (Creator or Admin)
GET /api/v1/credit-scoring/history/:creatorId - Get credit score history for a creator (Creator or Admin)
Application Status Endpoints
GET /api/v1 - Hello World endpoint
GET /api/v1/health - Health check endpoint
